## 1. 产品概述

基于TensorFlow.js和Canvas的纯前端汉字手写体生成应用。用户上传楷书或行书样本字图片，系统自动提取笔画顺序和轨迹，对目标文本生成具有相同风格的新手写体，支持多种风格融合与参数调节。

- 核心目标：让用户通过少量样本字即可生成大量风格统一的个人手写体
- 目标用户：书法爱好者、设计师、需要个性化字体设计用户
- 市场价值：提供无需后端的纯前端解决方案，保护用户隐私，降低使用门槛

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，直接使用 | 上传样本、生成手写体、调节参数、导出SVG |

### 2.2 功能模块
1. **首页：样本上传与预览
2. **笔画提取模块：骨架化、图搜索、轨迹提取
3. **风格融合模块：多风格加权平均
4. **生成控制模块：笔画粗细、速度、飞白效果调节
5. **预览与回放模块：逐笔动画回放、SVG导出
6. **文本生成模块：目标文本输入、批量生成

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 样本上传区 | 拖拽或点击上传楷书/行书样本字图片，支持多张上传 |
| 首页 | 笔画可视化区 | 显示提取的笔画骨架、顺序标注、轨迹预览 |
| 首页 | 参数调节区 | 滑块控制笔画粗细、书写速度、飞白强度 |
| 首页 | 风格融合区 | 多个样本风格权重调节 |
| 首页 | 文本输入区 | 输入目标文本，支持中英文混合输入 |
| 首页 | 生成预览区 | Canvas实时预览生成效果、SVG逐笔回放 |
| 首页 | 导出区 | 下载SVG矢量文件、复制SVG代码 |

## 3. 核心流程

用户上传样本字图片 → 系统进行图像预处理（二值化、去噪 → 骨架化提取笔画中线 → 图搜索确定笔画顺序和轨迹 → 建立笔画风格模型 → 用户输入目标文本 → 系统按笔画拼接或GAN生成 → 参数实时预览 → 逐笔动画回放 → 导出SVG矢量文件

```mermaid
    D --> E["建立风格模型"]
    E --> F["输入目标文本"]
    F --> G["笔画拼接/GAN生成"]
    G --> H["参数实时预览"]
    H --> I["逐笔动画回放"]
    I --> J["导出SVG文件"]
```

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深墨色(#1a1a1a)为主，搭配宣纸米白(#f5f0e6)背景，朱红(#c41e3a)点缀
- 按钮风格：圆角矩形，悬浮时有墨迹晕染效果
- 字体：使用书法风格字体搭配现代无衬线字体
- 布局风格：左右分栏，左侧控制面板，右侧预览区域
- 视觉元素：添加宣纸纹理背景，水墨晕染装饰

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|----------|
| 首页 | 样本上传区 | 拖拽上传框、缩略图网格、删除按钮 |
| 首页 | 笔画可视化区 | Canvas画布、颜色区分笔画、顺序数字标注 |
| 首页 | 参数调节区 | 滑块控件、数值显示、实时预览 |
| 首页 | 风格融合区 | 多滑块权重调节、风格预览卡片 |
| 首页 | 文本输入区 | 多行文本框、字数统计 |
| 首页 | 生成预览区 | Canvas画布、播放控制按钮、时间轴 |
| 首页 | 导出区 | 下载按钮、代码预览、格式选择 |

### 4.3 响应式
- 桌面端：左右分栏布局，控制区固定宽度，预览区自适应
- 移动端：上下堆叠布局，触摸友好的触控区域
- 触控优化：滑块区域增大，按钮最小尺寸44px
